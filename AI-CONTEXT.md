

## HARD GATE — Cross-site keyword registry (added 2026-07-06)

Before drafting, scheduling, or outlining ANY article on this site:

1. Read `docs/CROSS-SITE-KEYWORD-REGISTRY.md`.
2. Confirm the target keyword is reserved to THIS site in that file.
3. If the keyword is owned by another Arcadia site (HC / Rehab / ECC), or appears in the HARD BLOCKS table, or is unlisted: **STOP. Do not draft. Flag the conflict and wait for the keyword to be reserved in the Master Sheet.**
4. If the registry's "Generated" date is more than 7 days old, tell the user to run `npm run sync:registry` before trusting it.

This gate supersedes any softer keyword-check language elsewhere in this file or in docs/. "Differentiate by framing" is NOT an escape hatch — if the query cluster is owned elsewhere, the topic is owned elsewhere.
