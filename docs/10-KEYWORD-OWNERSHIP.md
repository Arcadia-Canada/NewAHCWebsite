# 10 — Keyword Ownership Map

Every target keyword must be owned by exactly **one page**. Two pages competing for the same keyword cannibalize each other and both lose. This doc is the single source of truth for which page owns which keyword.

Last updated: May 3, 2026

---

## Rules

1. **One page per keyword.** If two pages could target the same keyword, the page with higher topical authority wins. The other must differentiate (different angle, different long-tail) or be merged.
2. **Check before writing.** Before starting any new article, search this doc for the target keyword. If it's already assigned, either choose a different angle or flag a conflict.
3. **Update on publish.** When a new page goes live, add its keyword entry here. When a page is retired, remove or strikethrough its entry.
4. **Service pages own commercial keywords.** "Home care" + service type keywords belong to service pages, not articles.
5. **Condition pages own condition keywords.** "Care at home" + condition name keywords belong to condition pages.
6. **Articles own informational keywords.** "How to," "what to expect," "signs of," "guide" queries belong to resource articles.
7. **Location pages own geo keywords.** "[Service] + [city]" keywords belong to location pages.

## How to update this file

**When a new article publishes:**
1. Move its keyword entry from "Coming Soon" or add a new row to "Published resource articles."
2. Verify no other page targets the same primary keyword.

**When a conflict is resolved:**
1. Remove the entry from "Known conflicts" and add a note explaining the resolution.
2. Add the resolved article to the appropriate keyword section.

**When planning a new article not in the AUTHORITY-MAP:**
1. Check this doc first for the intended keyword.
2. If the keyword is already assigned, either differentiate or choose a new keyword.
3. Add the new keyword reservation to this doc before writing.
